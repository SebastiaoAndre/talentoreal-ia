# Roadmap Técnico — TalentoReal IA

## Fase 1 — Fundação (MVP Base)

- Estrutura do repositório
- Documentação oficial
- Definição da arquitetura
- Definição do modelo de dados
- Definição do fluxo do utilizador

## Fase 2 — Backend Inteligente

- Configuração Firebase
- Autenticação (Auth)
- Base de dados Firestore
- Estrutura de perfis (Talento / Empresa)
- Sistema de provas e validação
- Motor de Score de Talento

## Fase 3 — IA e Segurança

- Detecção de fraude em CV
- Validação automática de provas
- Matching inteligente Talento ↔ Empresa
- Sistema de reputação
- Proteção contra manipulação

## Fase 4 — Plataforma

- App Web (Next.js / FlutterFlow)
- Painel do Talento
- Painel da Empresa
- Sistema de candidaturas
- Notificações inteligentes

## Fase 5 — Expansão

- API pública
- Integração com LinkedIn / GitHub
- Sistema de certificação
- Escalabilidade
- Preparação para investidores
